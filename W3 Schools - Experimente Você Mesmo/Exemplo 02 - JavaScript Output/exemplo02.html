Possibilidades de exibição do JavaScript


Usando innerHTML

Para acessar um elemento HTML, você pode usar o método document.getElementById(id). Em seguida, use a propriedade innerHTML para alterar o conteúdo HTML do elemento HTML.


Usando innerText

Use innerText para alterar o texto interno do elemento HTML.

Use innerHTML quando quiser alterar um elemento HTML. Use innerText quando quiser alterar apenas o texto simples.


Usando document.write()

O método document.write() deve ser usado somente para testes. Usar document.write() depois que um documento HTML for carregado, excluirá todo o HTML existente.



Usando window.alert()

Em JS, o objeto window é um objeto de escopo global. Isso significa que variáveis, métodos e propreidades, por padrão, pertencem ao objeto window. Isso também significa que especificar a palavra-chave window é opcional.



Usando console.log()

Para fins de depuração, você pode chamar o console.log() no navegador para exibir dados.
