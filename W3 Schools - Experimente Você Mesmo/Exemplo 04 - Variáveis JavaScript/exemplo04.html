Variáveis JavaScript


Variáveis são contêineres para armazenamento de dados

É uma boa prática declarar variáveis antes de usá-las.

A palavra-chave 'var' deve ser usada somente em código escrito para navegadores antigos.


Quando usar var, let ou const?

  • Sempre declare variáveis
  • Sempre use 'const' se o valor não deve ser alterado
  • Sempre use 'const' se o tipo não deve ser alterado (arrays e objetos)
  • Use somente 'let' se não puder usar 'const'
  • Use somente 'var' se precisar oferecer suporte a navegadores antigos


Declarando uma variável JavaScript

Após declarar uma variável, ela não tem valor (tecnicamente é 'undefined'). É possível atribuir um valor ao declará-la.

É uma boa prática de programação declarar todas as variáveis no início de um script.

Se você colocar um número entre aspas, o restante dos números serão tratados como strings concatenadas.
